Welcome to SecretFlow PSI Library's documentation!
==================================================

This is the repo of Private Set Intersection(PSI) and Private Information Retrieval(PIR) from SecretFlow team, which provides PSI and PIR functionality for `SecretFlow <https://www.secretflow.org.cn/docs/secretflow>`_ , `SCQL <https://www.secretflow.org.cn/docs/scql/latest/zh-Hans>`_  and `SecretPad <https://www.secretflow.org.cn/docs/quickstart/mvp-platform>`_.

This repo is formerly psi/pir part from `secretflow/spu <https://github.com/secretflow/spu>`_ repo.
